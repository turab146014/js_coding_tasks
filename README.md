# JavaScript Practice Tasks

This repository contains 3 JavaScript practice tasks.  
These tasks helped me practice arrays, objects, functions, classes, loops, conditions, and string methods.

## Task 1: Student Result Analyzer

In this task, I used an array of student objects.  
I filtered passed students, calculated average marks, found the topper marks, and added grades based on marks.

Concepts used:
- Arrays
- Objects
- filter()
- for...of loop
- Functions
- Conditions

## Task 2: Product Discount System

In this task, I created a Product class.  
The class stores product name and price, calculates discount, and returns the final price after discount.

Concepts used:
- Classes
- Constructor
- Methods
- this keyword
- Objects

## Task 3: Word Frequency Counter

In this task, I counted how many times each word appears in a sentence.  
I also found the most repeated word.

Concepts used:
- Strings
- Objects
- Functions
- for...of loop
- for...in loop
- toLowerCase()
- split()
